# Agent Working Memory

Ao iniciar trabalho neste repositorio, leia primeiro:

- `docs/agent-memory/current-context.md`
- `docs/agent-memory/decisions.md`

Use estes arquivos como memoria operacional persistente do projeto. Antes de encerrar uma sessao relevante, atualize `current-context.md` com:

- o que mudou;
- decisoes tomadas;
- pendencias e proximos passos;
- comandos importantes executados;
- riscos ou bloqueios.

Nao salve segredos, tokens, chaves, senhas, dados pessoais sensiveis ou transcricoes completas de conversas. Prefira resumos acionaveis.
